# Contributing

Thank you for your interest in contributing to this project.

## Purpose

This repository explores practical applications of Artificial Intelligence
for Site Reliability Engineering (SRE) and DevOps use cases.

Contributions are welcome in areas such as:

- anomaly detection
- log analysis
- root cause analysis
- incident response workflows
- observability architectures
- documentation improvements
- reproducible lab examples

---

## How to Contribute

### 1. Fork the repository

Create your own fork and clone it locally.

### 2. Create a branch

Use a descriptive branch name.

Examples:

- `feature/add-anomaly-detection-lab`
- `docs/improve-incident-response-architecture`
- `fix/log-analyzer-output-format`

### 3. Make your changes

Please keep contributions focused and well-scoped.

### 4. Test your changes

Before opening a pull request, verify that:

- scripts run successfully
- documentation is updated when needed
- examples remain reproducible

### 5. Open a Pull Request

In your PR description, include:

- what changed
- why it changed
- how to test it
- any limitations or follow-up work

---

## Contribution Guidelines

### Code

- prefer readability over unnecessary complexity
- document assumptions when relevant
- keep examples reproducible
- avoid introducing hidden dependencies

### Documentation

Documentation should be:

- practical
- technically accurate
- easy to follow
- consistent with the project structure

### Labs and Examples

Each lab or example should ideally include:

- purpose
- architecture context
- prerequisites
- how to run
- expected output
- limitations
- future improvements

---

## Reporting Issues

When opening an issue, please describe:

- the problem
- expected behavior
- current behavior
- reproduction steps
- relevant logs or outputs

---

## Project Philosophy

This repository aims to be both:

- a learning resource
- a practical engineering sandbox

Contributions that improve clarity, reproducibility, and engineering value
are especially appreciated.
